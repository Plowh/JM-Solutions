Rainbow 6 F.A.Q

Frequently Asked Questions

Driver Failed to Load - How to Fix?

If your driver fails to load, try the following solutions:

  1. Run the BSOD Fix – Execute the bsod.bat fix provided.
  2. Disable Driver Signature Enforcement – Follow this tutorial for guidance.
  3. Permanently Disable Windows Defender – Ensure Windows Defender is turned off.
  4. Permanently Disable Windows Firewall – Make sure Windows Firewall is completely disabled.
  5. Update Your Drivers – Ensure all necessary drivers (GPU, chipset, etc.) are up to date.

Cheat Injects but Menu Doesn't Show Up

If you hear a beep after loading into the game but the menu does not appear, follow these steps:

  1. Run the Game on DX12 – Ensure that DirectX 12 is enabled.
  2. Check Your GPU Drivers – Your drivers might be outdated!
  3. Use Borderless Fullscreen Mode – Change your display mode in settings.
  4. For Windows 11 Users – Run the w11 fix.bat file.
  5. Verify Game Files – Use the game client to check for corrupted or missing files.
  6. Disable Virtualization – Ensure virtualization is turned off in your BIOS settings.
  7. Disable Core Isolation – Follow the instructions below.
  8. Disable Third-Party Overlays – Turn off overlays like Discord, NVIDIA GeForce Experience, and MSI Afterburner.
  9. Run as Administrator – Ensure both the game and the injector are running with admin privileges.

How to Disable Virtualization & Core Isolation?

Some cheats and applications require Virtualization and Core Isolation to be disabled. Follow these steps:

Disable Virtualization in BIOS:

  1. Restart your PC and enter the BIOS/UEFI (Usually F2, DEL, or ESC during boot).
  2. Look for Intel VT-x or AMD-V under Advanced, CPU Configuration, or Processor Settings.
  3. Disable it, then save and exit the BIOS.

Disable Core Isolation in Windows:

  1. Press Windows Key + S, type Core Isolation, and open Core Isolation Settings.
  2. Toggle Memory Integrity to Off.
  3. Restart your PC.

How to Disable Firewalls and Add Exception for loader.exe

If your firewall or antivirus is blocking the program, follow these steps:

Disable Windows Firewall:

  1. Open Control Panel and go to System and Security.
  2. Click Windows Defender Firewall.
  3. Select Turn Windows Defender Firewall on or off (left sidebar).
  4. Check Turn off Windows Defender Firewall for both Public and Private networks.
  5. Click OK.

Add an Exception for loader.exe in Windows Defender:

  1. Open Windows Security.
  2. Go to Virus & threat protection.
  3. Click Manage settings under Virus & threat protection settings.
  4. Scroll down and select Add or remove exclusions.
  5. Click Add an exclusion, choose File, and select loader.exe.

Adding Exception in Popular Antivirus Software:

For Avast:

  1. Open Avast Antivirus.
  2. Go to Menu > Settings > General > Exceptions.
  3. Click Add Exception and select loader.exe.

For Bitdefender:

  1. Open Bitdefender and go to Protection > Antivirus.
  2. Click Manage Exceptions and add loader.exe.

For Kaspersky:

  1. Open Kaspersky and go to Settings > Additional > Threats and Exclusions.
  2. Click Specify Trusted Applications and add loader.exe.

For Norton:

  1. Open Norton Antivirus and go to Settings > Antivirus.
  2. Click Scans and Risks, scroll to Exclusions, and add loader.exe.

If your exclusion isn't being added do this:

1. Run as Administrator

  • Open Windows Security → "Virus & threat protection" → "Manage settings" under "Virus & threat protection settings"
  • Scroll down to "Exclusions" and click "Add or remove exclusions"
  • If prompted, click "Yes" on the UAC prompt (if it doesn't show up, manually run Windows Security as an admin).

2. Check Group Policy / Registry Restrictions

  • If you're on a work/school PC, an admin might have blocked exclusions.
  • Check if Defender is managed:

Open Command Prompt as Admin and run:

reg query "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender"

If DisableExclusion is set to 1, exclusions are disabled.

3. Disable Tamper Protection (Temporarily)

  • Go to Windows Security → "Virus & threat protection" → "Manage settings"
  • Scroll down to "Tamper Protection" and turn it off
  • Now, try adding the exclusion again.

4. Add Exclusion via PowerShell

Try adding it manually with PowerShell (run as admin):

Add-MpPreference -ExclusionPath "C:\Your\Folder\Path"

or for a specific file:

Add-MpPreference -ExclusionProcess "YourFile.exe"

Then check:

Get-MpPreference | Select-Object -ExpandProperty ExclusionPath

5. Reset Windows Security

If nothing works, reset Defender settings:

Set-MpPreference -DisableRealtimeMonitoring $false

or reinstall with:

Get-AppxPackage Microsoft.SecHealthUI -AllUsers | Reset-AppxPackage

How to Enable DX12?

To enable DirectX 12:

  • Non-Steam Version: Click the down arrow next to the launch icon and select DX12.
  • Steam Version:
    1. Open Steam and right-click on Rainbow Six Siege.
    2. Go to Properties > General Tab > Set Launch Options.
    3. Add dx12 to the launch options.

If -dx12 isn't working on steam do this:

Step 1: Install Graphics Tools

  1. Open Settings → Go to Apps → Select Optional Features → Click Add a feature.
  2. Wait for the list to load. If the list doesn't load in a few minutes:
    • Press CTRL + SHIFT + ESC to open Task Manager.
    • Find "Settings", right-click it, and select End Task.
    • Repeat the previous step.
  3. Once the list loads, locate Graphics Tools, select it, and click Install.
  4. Wait for the installation process to complete.

Step 2: Open the DirectX Control Panel

  1. Press Windows Key + R to open the Run window.
  2. Type: 
    dxcpl
    and press Enter.

Step 3: Configure Direct3D Settings

  1. In the Direct3D 10/11/12 tab, click Edit List....
  2. Click the "..." button, browse for your game's main executable file, and click Add.
  3. Click OK to confirm the addition.
  4. Any additional executables you want to run with these settings can be added in the same manner.

Step 4: Adjust Device Settings

  1. Once the settings window becomes fully clickable, scroll down to Device Settings.
  2. Check "Disable Feature Level Upgrade".
  3. Choose a version next to "Feature level limit".
    • Make sure the version selected is supported by the game; otherwise, the game may fail to launch.
    • If the game gives a DirectX fatal error, consider selecting a different version.

Final Step: Apply and Launch

  1. Click OK to save your changes.
  2. Launch your game.

Program Closes After Connecting

If the program closes after reaching the connecting phase, follow these steps:

  1. Check Your System Date & Time:
    • Press Windows Key + S and type Date & Time settings.
    • Open the settings and ensure the correct time zone is selected.
    • Toggle Set time automatically to On.
    • If incorrect, manually set the date and time to the correct present date.
  2. Restart Your PC: Restarting ensures the changes take effect.
  3. Check for Windows Updates:
    • Go to Settings > Update & Security > Windows Update.
    • Click Check for Updates and install any pending updates.
  4. Run the Program as Administrator: Right-click on the executable file and select Run as Administrator.

Black Screen After Injection

If your screen goes black after injecting the cheat, try the following:

  1. Turn Off Shadows – Disable shadows in the game settings.
  2. Lower Graphics Settings – Reduce overall graphics settings to low.
  3. Disable Fullscreen Optimizations – Right-click on the game executable, go to Properties > Compatibility, and check Disable fullscreen optimizations.

"msvcp140.dll or msvcp141.dll Missing" Error

To fix this issue, install Microsoft Visual C++ Redistributable.

Discord64Hook Crash - How to Fix?

If you encounter the Discord64Hook crash, follow these steps:

  1. Run Registry Editor as Administrator – Press Windows Key + R, type regedit, and press Enter.
  2. Navigate to the following key: 
    Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\DiscordHookHelper64.exe
  3. Delete the Debugger Key – Right-click on the Debugger entry and select Delete.
  4. Restart Your System – Try launching the application again.
